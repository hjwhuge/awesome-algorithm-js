### 题目 - 有效的字母异位词

> [leetcode 242](https://leetcode-cn.com/problems/valid-anagram/)

给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词。

注意：若  s 和 t  中每个字符出现的次数都相同，则称  s 和 t  互为字母异位词。

示例  1:

```js
输入: (s = "anagram"), (t = "nagaram");
输出: true;
```

示例 2:

```js
输入: (s = "rat"), (t = "car");
输出: false;
```
